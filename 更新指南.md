## 🔄 更新流程

### 第一步：进入工作目录

```bash
cd /Users/wczzsmmb/chengxu/vibe/Pi/skills备份/temp-git-save
```

### 第二步：备份本地 Skills

```bash
# 删除旧的 skills 目录
rm -rf skills

# 从本地 Pi 环境复制最新的 skills
mkdir -p skills
for dir in ~/.pi/agent/skills/*/; do
  if [ -d "$dir" ] && [ ! -L "$dir" ]; then
    skill_name=$(basename "$dir")
    if [ -f "$dir/SKILL.md" ]; then
      cp -r "$dir" skills/
    fi
  fi
done
```

### 第三步：提交并推送

```bash
# 添加所有变更
git add .

# 提交
git commit -m "更新 skills 备份 $(date +%Y-%m-%d)"

# 推送到 GitHub
git push origin main
```

## ⚠️ 注意事项

1. **只操作 temp-git-save 目录** - 不要操作其他目录
2. **skills 目录会完全替换** - 每次更新都是全量替换
3. **符号链接不备份** - librarian 和 pdf 是 npm 包的符号链接，不需要备份
4. **推送前检查** - 用 `git status` 确认变更内容
