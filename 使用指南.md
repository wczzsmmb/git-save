# Pi Skills 使用指南

## 🎯 概述

本仓库包含 30 个 Pi Coding Agent 的 Skills，涵盖前端开发、后端开发、AI/ML、设计、文档等多个领域。

## 🚀 快速开始

### 第一步：克隆仓库

```bash
git clone https://github.com/wczzsmmb/git-save.git
cd git-save
```

### 第二步：安装 Skills

```bash
# 创建 skills 目录
mkdir -p ~/.pi/agent/skills

# 复制所有 skills
cp -r skills/* ~/.pi/agent/skills/

# 设置权限
chmod -R 755 ~/.pi/agent/skills/
```

### 第三步：配置符号链接（可选）

```bash
# 如果需要 npm 包的 skills
ln -sf ~/.pi/agent/npm/node_modules/pi-web-access/skills/librarian ~/.pi/agent/skills/librarian
ln -sf ~/.pi/agent/npm/node_modules/pi-pdf/skills/pdf ~/.pi/agent/skills/pdf
```

### 第四步：验证安装

```bash
# 列出所有 skills
ls -la ~/.pi/agent/skills/

# 在 Pi 中测试
# 运行: /test-skill librarian
```

## ❓ 常见问题

### Q: 安装后 skills 没有出现？
A: 检查目录权限：
```bash
chmod -R 755 ~/.pi/agent/skills/
```

### Q: 符号链接显示断开？
A: 确保 npm 包已安装：
```bash
cd ~/.pi/agent/npm
npm install
```

### Q: 如何更新到最新版本？
A: 参考 `更新指南.md`

### Q: 如何卸载某个 skill？
A: 删除对应目录：
```bash
rm -rf ~/.pi/agent/skills/skill-name
```
