# Pi Skills 更新指南

## 📋 概述

本指南用于更新 Pi Skills 备份仓库，当你安装了新的 skills 后，按照此流程更新备份并上传到 GitHub。

## 🔄 更新流程

### 第一步：备份本地 Skills

```bash
# 进入备份目录
cd /Users/wczzsmmb/chengxu/vibe/Pi/skills备份

# 运行备份脚本
./backup.sh
```

### 第二步：更新 Git 仓库

```bash
# 进入 Git 仓库目录
cd /Users/wczzsmmb/chengxu/vibe/Pi/skills备份/temp-git-save

# 复制最新的 skills
rm -rf skills
cp -r ../skills .

# 查看变更
git status

# 添加所有变更
git add .

# 提交变更
git commit -m "更新 skills 备份 $(date +%Y-%m-%d)"

# 推送到 GitHub
git push origin main
```

### 第三步：验证更新

```bash
# 检查 GitHub 仓库
# 访问 https://github.com/wczzsmmb/git-save 确认更新成功
```

## 📝 快速更新脚本

创建一个快捷脚本 `update.sh`：

```bash
#!/bin/bash
# Pi Skills 快速更新脚本

set -e

echo "🔄 开始更新 Pi Skills 备份..."

# 备份本地 skills
cd /Users/wczzsmmb/chengxu/vibe/Pi/skills备份
./backup.sh

# 更新 Git 仓库
cd temp-git-save
rm -rf skills
cp -r ../skills .

# 提交并推送
git add .
git commit -m "更新 skills 备份 $(date +%Y-%m-%d)"
git push origin main

echo "✅ 更新完成!"
```

## ⚠️ 注意事项

1. **备份前检查**：确保所有新安装的 skills 都在 `~/.pi/agent/skills/` 目录中
2. **符号链接**：librarian 和 pdf 是符号链接，备份脚本会自动处理
3. **提交信息**：使用有意义的提交信息，方便追踪变更
4. **定期更新**：建议每周更新一次备份

## 🔧 故障排除

### 问题：备份脚本找不到 skills
```bash
# 检查 skills 目录
ls -la ~/.pi/agent/skills/

# 确保目录存在
mkdir -p ~/.pi/agent/skills
```

### 问题：Git 推送失败
```bash
# 检查远程仓库配置
git remote -v

# 重新配置远程仓库
git remote set-url origin https://github.com/wczzsmmb/git-save.git
```

### 问题：符号链接备份失败
```bash
# 手动创建符号链接
ln -sf ~/.pi/agent/npm/node_modules/pi-web-access/skills/librarian ~/.pi/agent/skills/librarian
ln -sf ~/.pi/agent/npm/node_modules/pi-pdf/skills/pdf ~/.pi/agent/skills/pdf
```
