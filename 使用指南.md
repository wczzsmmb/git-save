# Pi Skills 使用指南

## 🎯 这是什么？

本仓库备份了 30 个 Pi Coding Agent 的 Skills，克隆后可以直接使用。

## 🚀 安装方法

### 第一步：克隆仓库

```bash
git clone https://github.com/wczzsmmb/git-save.git
cd git-save
```

### 第二步：复制 Skills 到 Pi

```bash
# 创建 skills 目录
mkdir -p ~/.pi/agent/skills

# 复制所有 skills
cp -r skills/* ~/.pi/agent/skills/

# 设置权限
chmod -R 755 ~/.pi/agent/skills/
```

### 第三步：验证安装

```bash
# 查看已安装的 skills
ls ~/.pi/agent/skills/
```

应该看到 30 个 skill 目录。
